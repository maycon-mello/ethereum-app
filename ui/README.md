# Typescript + React + Redux

# Scripts
```bash
yarn start
yarn build

yarn test
yarn test:watch
yarn test:karma
yarn test:karma:watch

# To run e2e tests you need to have the appllication running locally, `yarn start`
yarn test:e2e
yarn test:e2e:watch

yarn coverage
yarn coverage:watch

yarn storybook

yarn eslint
yarn eslint:watch

# Deploy to S3 bucket
yarn deploy

# Generate the type definition for redux modules
yarn generate-types
```
